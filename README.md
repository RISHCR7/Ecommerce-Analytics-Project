# 👟 Sneaker Resale Market Analysis Dashboard

## 📌 Project Overview
This project analyzes sneaker resale market data to uncover trends, profitability insights, and inventory performance.

It includes:
- Data analysis using Python
- Interactive dashboard using Streamlit
- Data storage using MySQL

- MySQL

---

## 🔍 Key Insights
- High-demand sneaker models generate most revenue  
- Online sales channels are more profitable  
- Longer inventory duration reduces profitability  
- Sales show trends over time
